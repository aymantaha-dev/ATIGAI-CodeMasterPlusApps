FAQ

Common questions about KinCode Plus.

Click any question to expand the answer.

Is KinCode Plus free?

Yes. The application is completely free to install and use.

How does pricing work?

You only pay your selected AI provider based on that provider's own pricing. KinCode Plus does not charge subscriptions or hidden fees.

Are API keys stored locally?

Yes. API keys are encrypted and stored locally in a protected environment on your device.

Does my code leave my computer?

Only when you explicitly request AI assistance. Relevant code is sent directly to your selected provider, and KinCode Plus does not proxy or store those requests.

Is it open source?

The repository is available on GitHub. Please review the published license to understand reuse and distribution conditions.

Is the Windows app digitally signed?

Digital signing is part of the release quality roadmap. Please review release notes for the latest status before managed deployment.

Will macOS/Linux versions be released?

Cross-platform support remains a long-term goal. Updates are shared through official release channels without fixed date commitments.
