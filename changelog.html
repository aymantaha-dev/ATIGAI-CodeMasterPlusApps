GitHub & AI Workflow Upgrade

Version 1.1.0 improves GitHub integration, adds a lightweight local Python runtime for AI workflows, and refines the full interface for better usability.

Improvements

  • Improved GitHub integration for easier linking, repository discovery, and faster project upload workflows
  • Enhanced AI integration with a limited local Python environment and curated libraries to reduce token usage and improve search, analysis, presentation, and editing tasks
  • Refined UI design, sectioning, and element organization to make the product cleaner and easier to use
v1.0.0 February 2026
Stable

Initial Release

Official first public release of Kincode+ — an AI-powered desktop development environment for Windows, Linux, and macOS.

New Features

  • Generate, modify, and refactor entire software projects from a single prompt
  • Advanced Project Tree View with deep folder structure support
  • Multi-file editing with professional-grade diff visualization
  • Dedicated AI Chat Panel with context-aware responses
  • Conversation History Log for tracking AI decisions
  • Comprehensive Project Analytics dashboard
  • Workspace Customization (font, tabs, auto-save, minimap)
  • Day & Night Mode with instant switching
  • Support for 15+ AI models (GPT, Claude, Kimi, DeepSeek, Gemini)
  • GitHub Integration for cloning repositories

Platforms

  • Windows: Full support with Setup.exe installer
  • macOS: Coming soon (Universal Binary)
  • Linux: Coming soon (AppImage & .deb)

Security

  • Local-first architecture
  • Local API key storage
  • No automatic cloud synchronization
  • Your code never leaves your machine
